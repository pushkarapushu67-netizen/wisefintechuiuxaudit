# Wise Fintech UX Audit

A deep-dive UX audit and product strategy case study on Wise (formerly TransferWise).

Built as an APM portfolio piece — covering pain point analysis, competitive benchmarking, RICE-prioritised recommendations, and a Point of View on the India NRI remittance opportunity.

**Research:** 286,000+ reviews analysed across App Store, Play Store, and Trustpilot.
**Output:** 3 product recommendations with full effort/impact scoring.
**Format:** Single-page interactive case study.
